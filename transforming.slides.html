This is the dataset used in the second chapter of Aurlien Gron's recent book 'Hands-On Machine learning with Scikit-Learn and TensorFlow'. It serves as an excellent introduction to implementing machine learning algorithms because it requires rudimentary data cleaning, has an easily understandable list of variables and sits at an optimal size between being to toyish and too cumbersome

The data contains information from the 1990 California census. So although it may not help you with predicting current housing prices like the Zillow Zestimate dataset, it does provide an accessible introductory dataset for teaching people about the basics of machine learning.

import numpy as np 

# mdia igual a 0 e um desvio padro igual a 1
def standardize(column):  
    z = (column - column.mean())/np.std(column)
    return z
  
total_bedrooms_standardized = standardize(housing.total_bedrooms)
fig, ax = plt.subplots(figsize=(15,10))
plt.hist(total_bedrooms_standardized)
plt.show()
  • Se a distribuio no Gaussiana ou o desvio padro muito pequeno, normalizar os dados uma escolha a ser tomada.

Muitas variveis na rea de biologia tm distribuio lognormal. Ento, analisar no a varivel coletada, mas a varivel transformada, isto , o logaritmo (decimal ou neperiano) dessa varivel, ajuda a estabilizar a varincia e tornar a distribuio normal. Mas como se reconhece a necessidade da transformao logartmica? Se a varincia dos grupos cresce com a mdia, razovel optar pela varivel transformada para a anlise. A explicao a de que, se a varivel que voc estuda resultante de vrios fatores que se multiplicam, ento essa varivel tem distribuio lognormal. Um exemplo a altura das rvores de um pomar. A altura de uma rvore afetada pelo solo, quantidade de gua, de luz solar, de ventos etc. Ento a varivel altura das rvores de um pomar tem distribuio lognormal. Tambm deve ter distribuio lognormal o tempo de sobrevivncia de bactrias em desinfetantes, o peso e a presso sangunea das pessoas.
